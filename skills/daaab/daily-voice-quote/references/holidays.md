# 節日與特殊日期清單

每日名言任務會在特殊節日時選擇應景名言，而非使用固定清單輪替。

## 極端天氣關鍵詞

當以下天氣現象出現時，名言應加入關懷提醒：

| 天氣 | 關懷方向 |
|------|---------|
| 颱風 / typhoon | 注意安全、堅強 |
| 暴雨 / heavy rain / 豪雨 | 帶傘、小心出行 |
| 暴風 / storm | 注意安全 |
| 暴雪 / heavy snow / blizzard | 保暖、注意出行 |
| 極端高溫 / heat wave | 多喝水、防曬 |
| 極端低溫 / cold wave / 寒流 | 保暖 |
| 地震 | 注意餘震、安全 |
| 空汙 / air quality 嚴重 | 戴口罩、減少外出 |

## 應景名言範例

### 春節
- 「一年之計在於春，一日之計在於晨。」
- 蘇軾：「人生到處知何似，應似飛鴻踏雪泥。」

### 中秋
- 蘇軾：「但願人長久，千里共嬋娟。」
- 李白：「舉頭望明月，低頭思故鄉。」

### 冬至
- 「冬至大如年。」
- 白居易：「邯鄲驛裡逢冬至，抱膝燈前影伴身。」

### 聖誕節
- Charles Dickens: "No one is useless in this world who lightens the burdens of another."
- Dr. Seuss: "Maybe Christmas doesn't come from a store."

> 💡 Agent 在執行時應根據節日氣氛自行挑選或創造應景的名言，不限於此清單。
