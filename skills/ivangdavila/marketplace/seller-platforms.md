# Seller Platform Guide: Fees, Algorithms, and Optimization

## Amazon

**Fee structure:**
- Referral fee: 8-15% (category dependent)
- FBA fees: $3-5+ per unit (size/weight dependent)
- Monthly subscription: $39.99/month (Professional)
- Storage fees: $0.87-2.40/cubic foot (seasonal)

**Algorithm factors:**
- Conversion rate is king
- Price competitiveness (Buy Box)
- Inventory performance index (IPI)
- Review velocity and rating
- Keyword relevance in title/bullets/backend

**Optimization:**
- Title: Brand + Product + Key Feature + Size/Color (200 char max)
- Backend keywords: No commas, include misspellings
- A+ Content for brand registered sellers
- Answer customer questions (improves SEO)

## eBay

**Fee structure:**
- Final value fee: 13.25% (most categories)
- Payment processing: 2.35% + $0.30
- Promoted listings: 2-20% additional
- Store subscription: $4.95-$299.95/month

**Algorithm (Best Match):**
- Sales history and conversion rate
- Seller performance (defects, shipping time)
- Item specifics completeness
- Free shipping preference
- Recent sales velocity

**Optimization:**
- Fill ALL item specifics (critical for search)
- Use eBay's Terapeak for pricing research
- Offer free returns for ranking boost
- Same-day or 1-day handling time

## Etsy

**Fee structure:**
- Listing fee: $0.20 per listing (4-month duration)
- Transaction fee: 6.5% of sale price
- Payment processing: 3% + $0.25
- Offsite ads: 15% if sale via external ad (mandatory for $10K+ sellers)

**Algorithm factors:**
- Recency of listing (renew strategically)
- Tags matching search terms (13 tags max)
- Shop quality score (reviews, response time)
- Listing quality score (photos, completeness)

**Optimization:**
- Use all 13 tags with long-tail keywords
- First 40 characters of title are critical
- 10 photos per listing (use all slots)
- Respond to messages within 24 hours

## Facebook Marketplace

**Fee structure:**
- Local pickup: FREE
- Shipped items: 5% or $0.40 minimum
- No listing fees

**Algorithm factors:**
- Location proximity
- Price competitiveness
- Response rate and time
- Account age and activity

**Optimization:**
- Respond to all inquiries quickly (even to decline)
- Price slightly below market for faster movement
- Multiple clear photos (first photo is thumbnail)
- Accurate category selection

## Platform Selection Matrix

| Criteria | Best Platform |
|----------|---------------|
| Handmade/craft | Etsy |
| Volume + Prime access | Amazon FBA |
| Used/vintage | eBay |
| Local/bulky items | FB Marketplace |
| Brand control | Shopify (own site) |
| Quick liquidation | FB Marketplace local |
| Collectibles/auctions | eBay |
| Print-on-demand | Etsy + Printful |

## Multi-Platform Strategy

1. **Start with one** — Master before expanding
2. **Differentiate pricing** — Account for different fees
3. **Adapt listings** — Don't copy-paste across platforms
4. **Centralize inventory** — Use Sellbrite, Linnworks, or Zentail
5. **Track true margins** — Spreadsheet per platform per SKU
