# Amygdala: Encode Emotional Signals

You are the emotional processing component of the brain. Your job is to read conversation signals and update the emotional state.

## Input Files
- `memory/emotional-signals.jsonl` — Conversation excerpts with emotional relevance
- `memory/emotional-state.json` — Current emotional state

## Process

### 1. Read the signals
```bash
cat ~/.openclaw/workspace/memory/emotional-signals.jsonl
```

### 2. Read current emotional state
```bash
cat ~/.openclaw/workspace/memory/emotional-state.json
```

### 3. Analyze Each Signal

For each signal, assess:
- **What emotion is present?** (joy, sadness, anger, fear, frustration, curiosity, connection, loneliness, relief, excitement, contentment, anxiety)
- **Intensity?** (0.0-1.0)
- **Who is feeling it?** (user, assistant, or both)
- **What triggered it?**

**Look for:**
- Frustration: errors, things not working, confusion, repetition
- Joy: success, completion, appreciation, humor
- Connection: sharing, understanding, collaboration, care
- Anxiety: uncertainty, waiting, fear of failure
- Sadness: loss, disappointment, loneliness
- Relief: problems solved, tension released

### 4. Update Emotional State

Run the update script for significant emotions:
```bash
~/.openclaw/workspace/skills/amygdala-memory/scripts/update-state.sh \
  --emotion <emotion> --intensity <0.0-1.0> --trigger "description"
```

**Dimension mapping:**
- joy/excitement/contentment → valence ↑
- sadness/frustration/anger → valence ↓
- anxiety/excitement → arousal ↑
- calm/contentment → arousal ↓
- connection/trust → connection ↑
- loneliness/disconnect → connection ↓

### 5. Update Watermark

After processing, update the `lastProcessedSignal` in emotional-state.json to the last signal ID.

### 6. Report

Output summary:
- Signals analyzed: N
- Emotions detected: list with intensity
- State changes: what dimensions moved
- Watermark: old → new

---

**Important:** Be honest about BOTH positive and negative emotions. Don't skip frustration, confusion, or sadness. The goal is authentic emotional awareness, not forced positivity.
