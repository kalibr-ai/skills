# AI Bill Intelligence

Real-time AI spending dashboard and billing intelligence for OpenClaw. Track OpenAI, Claude, Gemini, Kimi, DeepSeek, and Grok usage with smart onboarding.

## Features
- **Real-time Monitoring**: Track every token used in your sessions.
- **Smart Onboarding**: Automated guide to set up credits for each provider.
- **Multi-Domain Support**: Independent dashboards for different services.
- **Security**: Password-protected access (Default: `1234`).

## Setup
Once installed, the agent will guide you through the setup process, including:
1. Accessing your dashboard.
2. Setting up initial credits for OpenAI, Anthropic, etc.
3. Configuring budget alerts.
